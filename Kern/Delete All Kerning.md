## Delete All Kerning

**Description**
Deletes all kerning pairs across every master in the font. This is a global and irreversible operation intended for resetting kerning data.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## ⚠️ Warning

This action removes **all kerning data** from the font.
Make sure to save a backup before running.

---

## Features

* Deletes all kerning pairs
* Applies to all masters
* Counts total removed pairs
* Fast and efficient execution

---

## Core

* Iterates through all masters
* Clears kerning dictionaries
* Tracks number of removed pairs

---

## Scope

* Entire font
* All masters

---

## Requirements

* Glyphs App 3.x
* Open font

---

## Usage

1. Open your font
2. Run the script
3. Confirm deletion message

---

## Notes

* This action cannot be undone
* Does not affect glyphs, groups, or outlines
* Useful for resetting kerning from scratch
