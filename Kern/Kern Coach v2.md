## Kern Coach v2

**A guided kerning workflow system for precision and control**

Kern Coach is an advanced kerning assistant that helps you generate, analyze, and refine kerning pairs through a structured and interactive workflow.

It combines pair generation, spacing analysis, group logic, and tab-based visualization into a single environment designed for efficient kerning in complex typefaces.

---

## What it does

* Generates kerning pairs from structured glyph sets
* Guides kerning workflow using “King / Subdit” logic
* Displays real spacing (geometry-based margin)
* Applies kerning in controlled batches
* Manages kerning groups and exclusions
* Provides tools for testing, cleaning, and reviewing kerning

---

## Core concept

Kern Coach introduces a **hierarchical kerning model**:

* **Kings** → dominant glyphs (reference shapes)
* **Subdits** → dependent glyphs

This allows systematic kerning across large character sets while avoiding redundancy.

---

## ⚙️ Key Features

* Geometry-based spacing analysis (real outline distance)
* King/Subdit pair system
* Group-aware kerning logic
* Tab-based pair visualization
* JSON-based preset system (save/load sets)
* Context-aware prefixes and suffixes
* Filtering (exclude glyphs, groups, existing pairs)

---

## Tools

* **Kerning Groups Generator**
  Automatically builds or cleans kerning groups.

* **Manual Group Assignment**
  Assign groups to selected glyphs (LSB, RSB, or both).

* **Test Words Collections**
  Includes curated kerning test strings (Impallari, Cabarga, etc.).

* **Tab Tools**
  Generate tabs, inspect groups, and manage kerning directly.

---

## Quick Start

1. Define a set (Kings + Subdits)
2. Generate kerning pairs
3. Analyze spacing (margin + kern)
4. Apply kerning
5. Review results in tabs

---

## Why use it

Kerning large fonts manually is slow and inconsistent.
Kern Coach provides a **structured, repeatable workflow** that scales across full character sets while maintaining precision.
