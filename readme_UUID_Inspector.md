# UUID Inspector for GlyphsApp
Author: Josep Patau Bellart (programmed with AI assistance)  
License: MIT

A tool for GlyphsApp that lets you search for glyphs by UUID, character, name, or partial name, and displays detailed information. Includes kerning search based on UUIDs or names.

## Functionality

### Search by UUID
- Validate UUID format
- Locate the glyph inside the font
- Display glyph information
- Show kerning occurrences

### Search by Character or Name
- Search by exact glyph name
- Search by Unicode value
- Search by uniXXXX names
- Search partial name matches

### Kerning Search
Find occurrences in left and right kerning pairs.

## Glyph Information Provided
- Name
- UUID
- Unicode
- Category and subcategory
- Width
- Color
- Export status
- Character
- Kerning groups
- Bounding box dimensions

## Interface
Window with:
- UUID and character/name inputs
- Buttons: Search by UUID, Search by Character, Clear
- Tabs: Glyph, Kerning

## Installation
1. Open GlyphsApp.
2. Go to Script → Open Scripts Folder.
3. Add the .py script.
4. Reload scripts.

## Usage
1. Open a font.
2. Run Script → UUID Inspector.
3. Enter a UUID or character/name.
