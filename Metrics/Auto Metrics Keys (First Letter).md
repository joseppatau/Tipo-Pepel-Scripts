## Auto Metrics Keys (First Letter)

**Description**
Automatically assigns metrics keys to selected glyphs based on the first letter of their names, helping establish consistent spacing relationships quickly.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## Features

* Assign left, right, and width metrics keys
* Uses first letter of glyph name
* Works on selected glyphs
* Fast setup for spacing systems
* Undo support

---

## Core

* Glyph name parsing
* Automatic metrics key generation
* Batch assignment

---

## Scope

* Selected glyphs
* Metrics keys only

---

## Requirements

* Glyphs App 3.x
* Selected glyphs

---

## Usage

1. Select glyphs
2. Choose which keys to assign
3. Click **Run**

---

## Notes

* Only applies to glyphs starting with letters
* Uses format `=X` (Glyphs standard)
* Ideal for initial spacing setup
