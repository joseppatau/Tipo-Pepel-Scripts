## Clear Metrics Keys

**Description**
Removes metrics keys from selected glyphs, allowing you to break spacing dependencies and work with independent sidebearings.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## Features

* Remove left, right, and width metrics keys
* Selective removal by reference glyphs (e.g. H, O)
* Option to remove all metrics keys
* Works on selected glyphs only
* Undo support

---

## Core

* Glyph-level metrics key inspection
* Conditional removal based on user input
* Batch processing of selected glyphs

---

## Scope

* Selected glyphs
* Metrics keys only

---

## Requirements

* Glyphs App 3.x
* Selected glyphs

---

## Usage

1. Select glyphs
2. Enter reference glyphs (optional)
3. Choose removal mode
4. Click **Run**

---

## Notes

* Removes spacing dependencies between glyphs
* Useful before manual spacing adjustments
* Supports full undo (⌘Z)
