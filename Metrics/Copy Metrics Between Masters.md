## Copy Metrics Between Masters

**Description**
Copies spacing metrics and metrics keys from a selected source master to the current master, ensuring consistency across masters.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## Features

* Copy LSB, RSB, and width
* Copy metrics keys (left, right, width)
* Applies to all glyphs
* Works between any masters
* Undo support

---

## Core

* Layer-to-layer metrics transfer
* Master-based selection system
* Full glyph iteration

---

## Scope

* All glyphs
* Source → current master

---

## Requirements

* Glyphs App 3.x
* Multiple masters

---

## Usage

1. Select source master
2. Select target (current master)
3. Click **Copy Metrics**

---

## Notes

* Overwrites existing metrics in target master
* Includes both numeric values and key references
* Useful for maintaining consistency across design axes
