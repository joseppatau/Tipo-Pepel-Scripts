## Copy Metrics to Multiple Masters

**Description**
Copies spacing metrics and metrics keys from the active master to multiple selected masters, with flexible control over glyph scope and metric types.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## Features

* Copy metrics from active master to multiple masters
* Select which metrics to copy (LSB, RSB, width)
* Supports current glyph, selected glyphs, or entire font
* Includes metrics keys
* Multi-master selection UI
* Batch processing with feedback summary

---

## Core

* Master-to-master metrics distribution
* Scope-based glyph selection
* Selective metrics transfer

---

## Scope

* Current glyph / selected glyphs / all glyphs
* Multiple destination masters

---

## Requirements

* Glyphs App 3.x
* Multiple masters

---

## Usage

1. Select source master (active)
2. Choose destination masters
3. Select glyph scope
4. Choose metrics to copy
5. Click **Copy Metrics**

---

## Notes

* Does not overwrite source master
* Applies only to selected metrics
* Includes both numeric values and key references
* Supports full undo (⌘Z)
