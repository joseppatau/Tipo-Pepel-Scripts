# Copy instance name to Style Linking and append Italic

A GlyphsApp utility for copy instance name to style linking and append italic.

## Requirements

- Glyphs App 3.x
- macOS

## Usage

1. Open a font in Glyphs.
2. Select the relevant glyphs, layers, nodes, or master when the tool requires a selection.
3. Run **Copy instance name to Style Linking and append Italic** from the Scripts menu.
4. Review the options and the Macro Panel output before continuing with production files.
