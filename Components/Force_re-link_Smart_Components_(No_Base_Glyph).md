## Force Re-link Smart Components (No Base Glyph)

**Description**
Fixes disconnected smart components by recreating them across all glyphs and master layers.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
MIT

---

## Features

* Repairs broken smart component links
* Processes entire font automatically
* Works across all master layers
* Preserves position, scale, and rotation

---

## Core

* Component recreation
* Smart value detection
* Layer-wide iteration

---

## Scope

* All glyphs in font
* Master layers only
* Smart components only

---

## Requirements

* Glyphs App 3.x
* Open font

---

## Usage

1. Open a font
2. Run the script
3. Components are automatically fixed

---

## Notes

* Only affects components with smart values
* Replaces components to restore linkage
