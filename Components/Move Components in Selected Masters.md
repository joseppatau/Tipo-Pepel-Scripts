## Move Components in Selected Masters

**Description**
Moves components by a specified X/Y offset across selected masters for selected glyphs, using a simple directional UI.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## Features

* Move components in multiple masters simultaneously
* Directional controls (↑ ↓ ← →)
* Custom movement value
* Multi-master selection

---

## Core

* Component offsetting
* Master filtering
* Name-based component matching

---

## Scope

* Selected glyphs
* Selected masters
* Components in active layer as reference

---

## Requirements

* Glyphs App 3.x
* Open font

---

## Usage

1. Select glyph(s)
2. Run the script
3. Choose masters
4. Set movement value
5. Use arrow buttons to move components

---

## Notes

* Moves components matching by name across masters
* Stores last used value
* Includes undo grouping
