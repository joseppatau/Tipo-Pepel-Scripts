# Remove Smart Glyph Properties From All Glyphs

A GlyphsApp utility for remove smart glyph properties from all glyphs.

## Requirements

- Glyphs App 3.x
- macOS

## Usage

1. Open a font in Glyphs.
2. Select the relevant glyphs, layers, nodes, or master when the tool requires a selection.
3. Run **Remove Smart Glyph Properties From All Glyphs** from the Scripts menu.
4. Review the options and the Macro Panel output before continuing with production files.

## Important

This tool can make structural or irreversible changes. Save the font or work on a copy before running it.
