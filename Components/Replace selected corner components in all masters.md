# Corner Tool Pro (SMART SYNC FIX)

A GlyphsApp utility for corner tool pro.

## Requirements

- Glyphs App 3.x
- macOS
- Vanilla, included with current Glyphs installations

## Usage

1. Open a font in Glyphs.
2. Select the relevant glyphs, layers, nodes, or master when the tool requires a selection.
3. Run **Corner Tool Pro (SMART SYNC FIX)** from the Scripts menu.
4. Review the options and the Macro Panel output before continuing with production files.

## Safety improvements

- Validates every target shape and node before changing corner hints across masters.
- Restores the Glyphs interface even if an operation fails.
