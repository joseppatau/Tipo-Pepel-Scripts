## Enable / Disable Components Alignment in All Masters

**Description**
Enables or disables automatic alignment of components across all masters for selected glyphs, with options for selected or all components.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
MIT

---

## Features

* Enable or disable automatic alignment
* Works across all masters and special layers
* Option to affect selected or all components
* Undo support with grouped actions

---

## Core

* Component alignment toggle
* Multi-layer iteration
* Selection-aware processing

---

## Scope

* Selected glyphs
* All masters and special layers
* Selected or all components

---

## Requirements

* Glyphs App 3.x
* Open font

---

## Usage

1. Select glyphs
2. (Optional) Select components
3. Run the script
4. Choose enable or disable

---

## Notes

* “Selected components only” is enabled by default
* Includes undo grouping for safe operations
