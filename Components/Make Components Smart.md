## Make Components Smart (Glyphs 3 Safe)

**Description**
Converts components into smart components by generating smart axes and assigning interpolation values based on font masters.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
MIT

---

## Features

* Automatically creates smart component axes
* Maps master values to smart axes
* Assigns interpolation values to components
* Supports batch processing of multiple glyphs

---

## Core

* Smart axis generation
* Master value mapping
* Component value assignment

---

## Scope

* Selected glyphs
* Selected or all components
* Master layers only

---

## Requirements

* Glyphs App 3.x
* Open font with defined axes

---

## Usage

1. Select glyph(s)
2. (Optional) Select components
3. Run the script

---

## Notes

* Creates axes only if missing
* Uses font axes as reference
* Refreshes selection to update UI
