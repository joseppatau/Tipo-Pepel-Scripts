## Delete All Guidelines (All Masters)

**Description**
Removes all global and glyph-level guides across all masters in the font in a single fast operation.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## Features

* Deletes all global (master) guides
* Deletes all glyph-level guides
* Processes entire font in one run
* Optimized for speed (UI disabled during process)

---

## Core

* Global and local guide cleanup
* Batch processing across glyphs
* Fast execution mode

---

## Scope

* Entire font
* All masters
* All glyph layers

---

## Requirements

* Glyphs App 3.x
* Open font

---

## Usage

1. Open a font
2. Run the script
3. All guides are removed instantly

---

## Notes

* This action is destructive (cannot be undone per guide)
* Includes summary report of deleted guides
* Extremely fast even on large fonts
