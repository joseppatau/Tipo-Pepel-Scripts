## Align Nodes (Per Master)

**Description**
Aligns selected nodes to the extreme position (top, bottom, left, or right), with support for current or all masters.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## Features

* Align nodes up, down, left, or right
* Uses extreme node as reference (not average)
* Supports current master or all masters
* Works on selected nodes only
* Undo support

---

## Core

* Directional node alignment
* Extreme value detection (min/max)
* Multi-master processing

---

## Scope

* Selected nodes
* Current or all master layers

---

## Requirements

* Glyphs App 3.x
* Selected nodes

---

## Usage

1. Select nodes
2. Choose direction (↑ ↓ ← →)
3. Choose scope (current or all masters)
4. Apply alignment

---

## Notes

* Requires at least two selected nodes
* Maintains consistency across masters
* Ideal for cleaning outlines and ensuring precision
