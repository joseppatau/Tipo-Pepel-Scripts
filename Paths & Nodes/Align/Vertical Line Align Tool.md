## Vertical Line Align Tool

**Description**
Detects semi-vertical node groups and aligns them based on proxiApache2y and length criteria, improving structural consistency in glyph outlines.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## Features

* Detect semi-vertical node groups
* Group nodes by X proxiApache2y
* Minimum length filtering
* Align nodes to:

  * Left
  * Center
  * Right
* Detailed feedback on affected glyphs

---

## Core

* Node grouping by proxiApache2y
* Vertical span detection
* Alignment target calculation
* Batch correction system

---

## Scope

* Selected layers
* Active master

---

## Requirements

* Glyphs App 3.x
* Selected glyphs

---

## Usage

1. Select glyphs
2. Set margin and minimum length
3. Choose alignment mode
4. Click **Apply**

---

## Notes

* Designed for vertical consistency
* Useful for cleaning stems and structural lines
* Works best on structured glyphs
