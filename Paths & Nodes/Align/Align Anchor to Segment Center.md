## Align Anchor to Segment Center

**Description**
Aligns the selected anchor to the horizontal center of selected nodes across all masters, with optional italic compensation.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## Features

* Aligns anchor to center of selected nodes
* Works across all masters
* Optional italic angle compensation
* Simple UI with one-click apply

---

## Core

* Node averaging
* Italic-aware positioning
* Multi-master support

---

## Scope

* Selected anchor
* Selected nodes
* All masters

---

## Requirements

* Glyphs App 3.x
* Open font

---

## Usage

1. Select an anchor and nodes
2. Run the script
3. Click **Align Anchor to Selection**

---

## Notes

* Requires both anchor and nodes selected
* Italic compensation can be toggled
