## Prepolator Toolkit

**Description**
A comprehensive toolkit for preparing glyphs for interpolation, including node synchronization, anchor management, path correction, and structural validation across masters.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## Features

### 🔹 Node Synchronization

* Sync first node across masters
* Based on geometric extremes (north, south, east, west)

---

### 🔹 Anchor Management

* Reset anchors (auto-generate)
* Clear all anchors

---

### 🔹 Structure Cleanup

* Delete corner components
* Correct path direction
* Reorder paths consistently

---

### 🔹 Validation Tools

* Check node count across masters
* Detect interpolation mismatches

---

## Core

* Multi-master synchronization
* Structural normalization
* Interpolation safety checks

---

## Scope

* Current glyph
* All master and special layers

---

## Requirements

* Glyphs App 3.x
* Multi-master font

---

## Usage

1. Select a glyph
2. Choose a tool from the panel
3. Apply changes across masters

---

## Notes

* Designed for interpolation preparation workflows
* Helps prevent common variable font issues
* Some actions are destructive (anchors, corners)
