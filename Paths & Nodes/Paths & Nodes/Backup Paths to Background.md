## Copy to Background (All Masters)

**Description**
Copies the current outlines of all glyphs to their background layers across all masters, creating a visual backup for comparison and editing.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## Features

* Copies outlines to background layers
* Applies to all glyphs and all masters
* Fast batch processing
* Preserves current drawing state

---

## Core

* Layer duplication per master
* Background layer assignment
* Full font iteration

---

## Scope

* Entire font
* All masters

---

## Requirements

* Glyphs App 3.x
* Open font

---

## Usage

1. Open font
2. Click **Copy ALL to background**
3. Use background layers for reference

---

## Notes

* Overwrites existing background layers
* Does not affect foreground outlines
* Useful before applying global changes
