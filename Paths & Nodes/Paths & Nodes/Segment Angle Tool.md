## Segment Angle Tool

**Description**
Adjusts angles between node pairs using geometric constraints, allowing precise control over segment direction and consistency.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## Features

* Apply custom angle between node pairs
* Direction control (/ and )
* Axis-based or automatic alignment
* Range-based processing (X or Y)
* Distance threshold filtering
* Option to ignore handles

---

## Core

* Angle-based geometric transformation
* Node pair detection within range
* Axis-constrained adjustment system

---

## Scope

* Current layer
* Selected nodes or nodes in range

---

## Requirements

* Glyphs App 3.x
* Node selection or defined range

---

## Usage

1. Set desired angle
2. Choose direction and axis
3. Define range (optional)
4. Click **Apply**

---

## Notes

* Moves one node relative to another
* Ignores very small segments
* Designed for precise geometric corrections
* Useful for diagonal consistency and stylistic control
