## Transform (All Masters) – TURBO

**Description**
Applies Glyphs-style transformations (translate, scale, slant) across current, all, or selected masters with high performance and fine control.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## Features

* Translate (X / Y)
* Scale (independent horizontal and vertical values)
* Slant (angle-based transformation)
* Apply to:

  * Current master
  * All masters
  * Selected masters (custom selection)
* High-performance batch processing
* Persistent settings between sessions

---

## Advanced Controls

* Select specific masters via checkbox list
* “Select All / Deselect All” toggle
* Dynamic UI that expands based on master selection
* Saved preferences using Glyphs defaults

---

## Core

* Node-level transformation engine
* Anchor transformation support
* Sequential transform pipeline:

  * Translate → Scale → Slant
* Optimized iteration across glyphs and layers

---

## Scope

* Selected glyphs
* Entire font (if no selection)
* Master layers and special layers

---

## Requirements

* Glyphs App 3.x

---

## Usage

1. Select glyphs (optional)
2. Define transformation values:

   * Translate
   * Scale
   * Slant
3. Choose scope:

   * Current Master
   * All Masters
   * Selected Masters
4. Click **OK**

---

## Notes

* Works as an extended version of the native Glyphs transform panel
* Adds multi-master and selective master control
* Designed for large-scale transformations across families
* Optimized for speed and production use
