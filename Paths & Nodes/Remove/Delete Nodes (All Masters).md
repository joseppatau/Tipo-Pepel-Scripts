## Delete Nodes (All Masters)

**Description**
Deletes selected nodes or their handles across all master layers, ensuring structural consistency in multi-master fonts.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## Features

* Delete selected nodes across all masters
* Remove handles only (keep main nodes)
* Maintains path and node index consistency
* Safe multi-master processing
* Undo support

---

## Core

* Node index mapping between masters
* Path-based structural consistency
* Selective deletion logic

---

## Scope

* Selected nodes
* All master layers of the glyph

---

## Requirements

* Glyphs App 3.x
* Selected nodes

---

## Usage

1. Select nodes in current master
2. Choose:

   * Delete nodes
   * Delete handles only
3. Apply to all masters

---

## Notes

* Only applies to master layers
* Requires consistent path structure
* Essential for variable font workflows
