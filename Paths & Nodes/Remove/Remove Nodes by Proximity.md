## Remove Nodes by ProxiApache2y

**Description**
Removes nodes that are closer than a defined distance, simplifying outlines and reducing unnecessary complexity.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## Features

* Detect nodes within a proxiApache2y threshold
* Remove overlapping or redundant nodes
* Adjustable margin value
* Works on selected glyphs
* Undo support

---

## Core

* Distance-based node comparison
* Duplicate node detection
* Safe node removal

---

## Scope

* Selected glyphs
* Current master

---

## Requirements

* Glyphs App 3.x
* Glyph selection

---

## Usage

1. Select glyphs
2. Set margin value
3. Click **Remove Nodes**

---

## Notes

* Larger margin removes more nodes
* May affect curve fidelity
* Ideal for cleaning imported or noisy outlines
