## MasterSync Glyphs

**Description**
Copies shapes, spacing, and metrics from a selected source master to the current master for selected glyphs, ensuring consistency across masters.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## Features

* Copy shapes between masters
* Copy width, LSB, and RSB
* Copy metrics keys (left/right)
* Works on selected glyphs only
* Interactive UI with master selection

---

## Core

* Master-to-master layer mapping
* Shape duplication system
* Metrics and spacing synchronization

---

## Scope

* Selected glyphs
* Source → current master

---

## Requirements

* Glyphs App 3.x
* Multiple masters
* Selected glyphs

---

## Usage

1. Select glyphs
2. Choose source master
3. Run **Apply Copy**
4. Review results in current master

---

## Notes

* Overwrites existing shapes in target master
* Preserves glyph structure
* Ideal for synchronizing designs across masters
