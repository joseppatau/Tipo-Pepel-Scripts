## Move Elements in Masters

**Description**
Moves paths (nodes), components, or both across the current or all masters using a simple directional interface.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## Features

* Move nodes, components, or both
* Works in current master or all masters
* Directional controls (↑ ↓ ← →)
* Custom movement value

---

## Core

* Node and component offsetting
* Layer targeting
* Selection-based movement

---

## Scope

* Selected glyphs
* Selected elements (nodes/components)
* Current or all masters

---

## Requirements

* Glyphs App 3.x
* Open font

---

## Usage

1. Select glyph(s)
2. Select nodes and/or components
3. Run the script
4. Choose mode and scope
5. Use arrow buttons to move

---

## Notes

* Moves only selected nodes or components
* Supports whole path movement if path is selected
* Includes undo grouping
