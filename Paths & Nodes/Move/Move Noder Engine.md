## Move Noder Engine

**Description**
Advanced tool for moving and adjusting node positions and distances using axis-based constraints, with support for multiple glyph scopes and geometric rules.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## Features

### 🔹 Single Node Movement

* Move nodes to a new X or Y position
* Based on tolerance range
* Multiple scope options:

  * Current glyph
  * Selected glyphs
  * Tab glyphs
  * Entire master

---

### 🔹 Multi-Node Distance Control

* Detect node pairs by distance
* Adjust spacing between nodes
* Multiple movement modes:

  * Both sides
  * First node only
  * From center
  * Second node only

---

### 🔹 Smart Constraints

* Optional liApache2s based on:

  * Baseline
  * x-height
  * Cap height
* Prevents invalid geometry changes

---

### 🔹 Advanced Processing

* Unique node pairing system
* Tolerance-based filtering
* Batch processing across glyphs

---

## Core

* Axis-based node transformation
* Distance evaluation system
* Constraint-aware adjustments
* Multi-scope glyph processing

---

## Scope

* Current glyph / selected / tab / entire master
* Active master layer

---

## Requirements

* Glyphs App 3.x
* Editable outlines

---

## Usage

1. Choose tab:

   * Single node
   * Multiple nodes
2. Set axis and values
3. Define scope
4. Apply transformation

---

## Notes

* Designed for precise geometric control
* Useful for spacing, stems, and structural adjustments
* Supports large-scale transformations
