## Shadow Engine

**Description**
Generates shadow shapes by offsetting glyph outlines and subtracting the original shapes, creating clean shadow effects.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## Features

* Create shadows using X/Y offset
* Boolean subtraction for clean results
* Works on selected glyphs
* Automatically cleans overlaps
* Simple and fast UI

---

## Core

* Affine transformation (offset)
* Boolean path subtraction
* Component decomposition
* Path cleanup and correction

---

## Scope

* Selected glyphs
* Current master
* Path-based shapes

---

## Requirements

* Glyphs App 3.x
* Open font

---

## Usage

1. Select glyph(s)
2. Set X and Y offset
3. Run the script
4. Apply shadow

---

## Notes

* Removes non-master layers before processing
* Decomposes components automatically
* Uses original shapes as subtraction mask
* Result replaces original paths
