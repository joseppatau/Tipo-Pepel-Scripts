# Genera accents dels glifs seleccionats (DEBUG)

A GlyphsApp utility for genera accents dels glifs seleccionats.

## Requirements

- Glyphs App 3.x
- macOS

## Usage

1. Open a font in Glyphs.
2. Select the relevant glyphs, layers, nodes, or master when the tool requires a selection.
3. Run **Genera accents dels glifs seleccionats (DEBUG)** from the Scripts menu.
4. Review the options and the Macro Panel output before continuing with production files.
