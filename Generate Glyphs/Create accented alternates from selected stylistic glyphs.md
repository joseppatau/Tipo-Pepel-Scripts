# Create accented alternates from selected stylistic glyphs

A GlyphsApp utility for create accented alternates from selected stylistic glyphs.

## Requirements

- Glyphs App 3.x
- macOS

## Usage

1. Open a font in Glyphs.
2. Select the relevant glyphs, layers, nodes, or master when the tool requires a selection.
3. Run **Create accented alternates from selected stylistic glyphs** from the Scripts menu.
4. Review the options and the Macro Panel output before continuing with production files.
