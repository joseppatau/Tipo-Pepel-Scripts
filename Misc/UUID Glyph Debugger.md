## UUID Glyph Debugger

**Description**
Advanced inspection tool for exploring glyph data, including UUID lookup, character search, and kerning relationship analysis.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
MIT

---

## Features

* Search glyphs by UUID
* Search by character or glyph name
* Partial name matching
* Display detailed glyph information
* Inspect kerning relationships
* Dual-tab interface (Glyph / Kerning)

---

## Core

* UUID validation and lookup
* Unicode and name-based search
* Kerning dictionary inspection
* Safe glyph data extraction

---

## Scope

* Entire font
* Active master (kerning view)

---

## Requirements

* Glyphs App 3.x
* Open font

---

## Usage

1. Enter UUID or glyph name/character
2. Click search
3. Review glyph info and kerning data

---

## Output

* Glyph metadata (name, UUID, Unicode, category)
* Dimensions and metrics
* Kerning groups
* Kerning pairs involving the glyph

---

## Notes

* UUID search requires valid format
* Partial name search supported
* Useful for debugging complex fonts
