## Live Glyph Preview Engine

**Description**
An advanced real-time glyph preview system with axis controls, interpolation, zoom, and interactive navigation for GlyphsApp scripts.

**Author**
Josep Patau Bellart (with AI assistance)

**License**
Apache2

---

## Features

* Real-time glyph rendering
* Axis-based interpolation controls
* Zoom and pan (drag interaction)
* Automatic glyph detection
* Multi-master support
* Live updates with timer refresh

---

## Interactive Controls

* Axis sliders with live interpolation
* Manual value input
* Hide/show axes
* Zoom slider and reset
* Mouse dragging for navigation

---

## Core

* Node-wise interpolation engine
* Layer selection and fallback logic
* Timer-based redraw system
* Path reconstruction (Bezier + manual)
* Cached rendering system

---

## Scope

* Current glyph
* All masters and interpolated instances

---

## Requirements

* Glyphs App 3.x
* Variable font with axes (optional but recommended)

---

## Usage

1. Open the panel
2. Select a glyph
3. Adjust axis sliders
4. Navigate with zoom and drag
5. Preview interpolation in real time

---

## Notes

* Includes custom interpolation fallback
* Automatically tracks active glyph
* Optimized for interactive workflows
* Suitable for debugging variable fonts
